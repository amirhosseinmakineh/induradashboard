# AngularJS (1.8) SPA Starter (UI-Router)

✅ No backend / No API calls.  
✅ Components + controllers + routing only.  
✅ Services are created but intentionally empty (stubs).

## Run
Use any static server:

### Option A: VSCode Live Server
- Open `index.html` -> "Open with Live Server"

### Option B: Node http-server
```bash
npx http-server -p 8080 .
```
Then open: http://localhost:8080

## Routes
- `/` Home (buttons to navigate)
- `/auth/login`
- `/auth/register`
- `/dashboard` (layout with sidebar)
- `/dashboard/users` (User Management)

## Notes
- Uses UI-Router (recommended for AngularJS routing).
- Uses `controllerAs` syntax everywhere.
